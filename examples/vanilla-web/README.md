# Vanilla JS Example

Demonstrates `@auser/workflow-graph-web` with plain HTML + ES modules.

## Features shown

- Dark / light / high-contrast theme switching
- Minimap toggle
- Layout direction toggle (left-to-right / top-to-bottom)
- Edge click and node selection callbacks
- Auto-resize

## Run

```bash
npm install @auser/workflow-graph-web
npx serve .
```

Open `http://localhost:3000` in your browser.
